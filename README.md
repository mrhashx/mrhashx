## 🔬 Featured Academic & Research Projects

### 🏥 1. [Brain Tumor Classification via Fine-Tuned CNNs](https://github.com/mrhashx/brain-tumor-classification-deep-learning)
An advanced medical computer vision pipeline utilizing transfer learning on VGG16, ResNet50, and InceptionV3 architectures to classify brain MRI scans into multi-class tumor categories (Glioma, Meningioma, Pituitary) and healthy controls. Features multi-stage frozen backbone stabilization and low-learning-rate fine-tuning.

### 🖼️ 2. [Digital Image Processing (DIP) Algorithms & Filters Toolkit](https://github.com/mrhashx/digital-image-processing-toolkit)
A mathematically rigorous image engineering toolkit developed from scratch in Python. Implements custom pixel-level point transformations, manual 2D convolution engines with dynamic boundary padding modes, and 2D Discrete Fourier Transform (DFT) frequency lowpass filters.

### 🏡 3. [Ames Housing Price Prediction Pipeline](https://github.com/mrhashx/ames-housing-regression-analysis)
A comprehensive machine learning regression framework optimizing high-dimensional feature spaces on the Ames Housing dataset. Features log-transformed non-linear target scaling, ColumnTransformer data preprocessors, and deep benchmarking across embedded and filter-based feature selection methods (Lasso, Adaptive Lasso, Chi-Square).

### 💬 4. [Cyberbullying & Offensive Language Detection Suite](https://github.com/mrhashx/cyberbullying-detection-nlp)
An end-to-end NLP filtering engine built to secure social platform comment metadata. Implements PorterStemmer token normalization, unigram/bigram sublinear TF-IDF vectorizers, and maximum-margin text classification engines benchmarking LinearSVC, Logistic Regression, and Multinomial Naive Bayes.
